# 旧版重校准说明

2026-07 前的项目保留了若干有价值思想：Pydantic 契约、oracle 隔离、evidence、报告构造和 synthetic 扰动。但其活动叙事跨越 manual JSON、CARLA、nuScenes、nuPlan 和 SafeBench；感知/规划/控制节点大多是顺序图中的固定 scorer，历史表格和媒体也无法证明一条真实控制链。

本次重构从活动仓库中移除了这些文件、生成视频、失效 CARLA 脚本、数据集 adapter、论文 PDF 与宽泛结果声明。旧树保留在 `legacy-before-openpilot-recalibration` 分支，但不再作为新 openpilot-like workflow 的证据。
