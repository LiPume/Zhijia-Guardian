# Legacy recalibration

The pre-2026-07 project retained good ideas—Pydantic contracts, oracle isolation, evidence, report construction and synthetic perturbations—but its active story spanned manual JSON, CARLA, nuScenes, nuPlan and SafeBench. Perception/planning/control nodes were mostly fixed scorers in a sequential graph, and historical tables/media could not demonstrate one real control chain.

Those files, generated videos, stale CARLA scripts, dataset adapters, paper PDFs and broad result claims were removed from the active repository during this refactor. The `legacy-before-openpilot-recalibration` branch preserves the old tree. They are deliberately not presented as evidence for the new openpilot-like workflow.
